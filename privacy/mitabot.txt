Доступ к данным
Бот использует минимальный набор данных, необходимый для выполнения своих функций. 

Конфиденциальность
Все сообщения и данные пользователей остаются в пределах чата. Бот не передаёт информацию третьим лицам. ( сделки меньше 100 долларов не принимаем ) 

Хранение данных
Бот не сохраняет и не обрабатывает данные за пределами текущей сессии. ( пиздёж ) 

Ограничение доступа
Бот доступен только для использования в чате KomaruChatting.

Обновления и исправления
Все обновления проверяются на безопасность перед развертыванием.

Контакты
По вопросам безопасности обращайтесь к администратору.
